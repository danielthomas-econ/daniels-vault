Typical CSS code for a picture looks like this:
~~~css
img{
  display: block;
  margin-left: auto;
  margin-right: auto;
  width: 50%;
}
~~~
While this isn't very intuitive, the `margin-left/right` arguments set to `auto` will nearly center the image.
